## About Allow Copy for Safari

Allow Copy for Safari removes copy-paste restrictions on websites that disable text selection on macOS. Some websites use JavaScript to prevent copying, disable right-click context menus, or block text selection entirely. This extension restores your ability to copy text from any webpage, bypassing those restrictions.

The ability to copy text from a webpage is a fundamental browsing action. When websites block this capability — whether to protect content or prevent scraping — it frustrates users who have legitimate reasons to copy information. Allow Copy restores this basic functionality with a simple toggle.

Enable the extension globally or toggle it per site to maintain control over when copy-paste restrictions are bypassed. Re-enable right-click context menus that some sites disable, restoring full browser functionality for every website you visit.
