## About Command Palette

Command Palette brings an Arc browser-style command palette to Safari on macOS. Quickly search tabs, bookmarks, history, and perform browser actions using a fast keyboard-driven interface — no mouse needed. If you love the command palette experience from Arc, VS Code, or Spotlight, this extension brings that power to Safari.

Keyboard-driven navigation is dramatically faster than clicking through menus and tabs. Open the command palette with a shortcut, type a few characters, and instantly jump to any open tab, bookmark, or history entry. Browser actions like opening new tabs, managing windows, and accessing settings are all available through the same unified interface.

Customizable keyboard shortcuts let you tailor the experience to your workflow. The fuzzy search finds what you need even with partial or approximate typing, making navigation feel effortless and instantaneous.

## Who is Command Palette for?

- Power users who prefer keyboard-driven navigation over mouse clicking
- Former Arc browser users who want similar functionality in Safari
- Developers accustomed to command palettes in code editors
- Anyone with many open tabs who needs fast tab switching
- Mac users who want to speed up their Safari workflow
