## About Color Palette

Color Palette is a Safari extension for macOS that extracts the complete color scheme from any website. Designers and developers can instantly identify hex codes, RGB values, and HSL values for every color used on a page, building inspiration palettes from their favorite sites.

Finding the exact colors used on a well-designed website usually requires digging through CSS or using separate color picker tools. Color Palette automates this by analyzing the entire page and presenting a comprehensive palette of all colors used — backgrounds, text, accents, and more.

Export your extracted palettes for use in design tools, or save and organize color collections for future reference. Whether you are building a design system, seeking color inspiration, or verifying brand consistency, this extension makes color discovery instant and precise.

## Who is Color Palette for?

- Designers seeking color inspiration from well-designed websites
- Developers implementing designs who need exact color values
- Brand managers verifying color consistency across web properties
- UI/UX designers building color palettes for new projects
- Anyone curious about the colors behind their favorite websites on Mac

## Why choose Color Palette?

Color Palette provides instant, comprehensive color extraction that would take significant manual effort otherwise. Every color on the page is identified with exact values in multiple formats — hex, RGB, and HSL — ready for immediate use.

The ability to save and organize color collections makes it more than a one-time tool. Build a library of color palettes from across the web to reference and reuse in your design work on macOS.
