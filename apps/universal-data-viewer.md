## About Universal Data Viewer

Universal Data Viewer is a Safari extension for macOS that renders structured data files beautifully in your browser. View Markdown, JSON, XML, CSV, and other data formats with proper formatting, syntax highlighting, collapsible sections, and navigation — turning raw data into readable documents.

When you open a raw Markdown file, JSON response, XML document, or CSV dataset in Safari, you normally see unformatted text that is difficult to read and navigate. Universal Data Viewer detects these file types and automatically renders them with appropriate formatting, making the data immediately useful.

Syntax highlighting for code files, collapsible sections for deeply nested structures, and search functionality for large files make this extension invaluable for anyone who regularly works with structured data in the browser.

## Who is Universal Data Viewer for?

- Developers viewing Markdown documentation, JSON APIs, and XML feeds
- Data analysts examining CSV datasets and structured data files
- Technical writers previewing Markdown documents in the browser
- Anyone who encounters raw data files while browsing
- Mac users wanting better rendering of structured files in Safari

## Why choose Universal Data Viewer?

Universal Data Viewer handles multiple data formats in a single extension, eliminating the need for separate Markdown viewers, JSON formatters, and XML renderers. The automatic format detection means it works without any configuration.

Syntax highlighting, collapsible navigation, and search make even large, complex files manageable. It is the most versatile data viewing extension available for Safari on macOS.
