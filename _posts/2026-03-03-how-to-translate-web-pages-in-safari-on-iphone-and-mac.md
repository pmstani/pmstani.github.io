## Breaking the Language Barrier in Safari

The internet is a global resource, but language barriers can lock you out of valuable content. Whether you are reading international news, researching products on foreign e-commerce sites, or exploring academic papers written in other languages, you need a reliable way to translate web content without leaving Safari.

While Safari offers basic built-in translation for a limited set of languages, a dedicated translation extension provides far more flexibility, accuracy, and control. Here is how to set up seamless web page translation on both iPhone and Mac.

## Using Translator Safari Extension

[Translator Safari Extension](/apps/translator-safari-extension/) is purpose-built for translating web content within Safari. It supports dozens of languages, detects the source language automatically, and lets you translate either entire pages or just selected text passages.

## How to Translate an Entire Web Page

### On Mac

1. **Install the extension.** Go to Safari > Settings > Extensions, click More Extensions, and search for Translator Safari Extension. Install and enable it.
2. **Visit a foreign-language page.** Navigate to any website in a language you want to translate.
3. **Click the translate icon.** Click the Translator Safari Extension icon in the Safari toolbar.
4. **Select your target language.** Choose the language you want the page translated into.
5. **View the translated page.** The entire page content will be replaced with the translated version, maintaining the original layout and formatting.

### On iPhone and iPad

1. **Install and enable.** Go to Settings > Apps > Safari > Extensions, install Translator Safari Extension, and toggle it on.
2. **Open a foreign-language page** in Safari.
3. **Tap the extensions menu** (the puzzle piece icon or the aA button in the address bar).
4. **Select Translator Safari Extension** and choose your target language.
5. **Read the translated page** directly in Safari.

## How to Translate Selected Text Only

Sometimes you do not need the entire page translated. Maybe you are reading a bilingual document, or you only need to understand a specific paragraph. Translator Safari Extension handles this too:

1. **Select the text** you want to translate by clicking and dragging (Mac) or tap-and-hold to select (iPhone).
2. **Right-click** (Mac) or tap the contextual menu (iPhone) and choose the translate option.
3. **View the translation** in a popup that appears near your selection.

This selective translation is perfect for quickly checking unfamiliar words or sentences without translating content you can already read.

## Supported Languages

Translator Safari Extension supports a wide range of languages, including but not limited to:

- **European:** English, Spanish, French, German, Italian, Portuguese, Dutch, Polish, Russian, Ukrainian, Swedish, Norwegian, Danish, Finnish, Greek, Czech, Romanian, Hungarian
- **Asian:** Chinese (Simplified and Traditional), Japanese, Korean, Vietnamese, Thai, Hindi, Indonesian, Malay
- **Middle Eastern:** Arabic, Hebrew, Turkish, Persian
- **African:** Swahili, Amharic

Automatic language detection means you never need to manually specify the source language. The extension identifies it and translates directly to your chosen target language.

## Tips for Better Translations

**Use page translation for long content.** When reading an entire article or product listing, translate the whole page for a seamless reading experience.

**Use text selection for quick lookups.** For individual words or sentences, selected-text translation is faster and less disruptive.

**Compare with the original.** If a translation seems unclear, you can toggle back to the original text to see the source content and interpret the translation in context.

**Set a default target language.** If you primarily translate into one language, set it as the default in the extension settings to save a step each time.
