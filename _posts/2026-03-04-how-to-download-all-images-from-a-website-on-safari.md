## Why You Need a Bulk Image Downloader for Safari

Downloading images from websites one by one is tedious. Right-click, save image, choose a folder, repeat. If you need to download a handful of product photos, a gallery of reference images, or visual assets from a design inspiration site, that manual process can eat up a surprising amount of time.

A bulk image downloader for Safari solves this problem by detecting every image on a web page and letting you download them all at once, or select specific ones to save. Here is how to set it up and use it effectively.

## Getting Started with Image Downloader

[Image Downloader](/apps/image-downloader/) is a Safari extension that scans any web page and presents you with all the images it contains. You can preview, filter, and download images individually or in bulk.

## How to Download All Images from a Web Page

### Step 1: Install Image Downloader

On Mac, open Safari and go to **Safari > Settings > Extensions**. Click **More Extensions**, search for Image Downloader, and install it. Enable the extension in the Extensions tab.

On iPhone, navigate to **Settings > Apps > Safari > Extensions**, find Image Downloader in the App Store, install it, and toggle it on.

### Step 2: Navigate to the Page with Images

Open the web page containing the images you want to download. This could be a photo gallery, a product listing page, a design portfolio, or any page with images you need.

### Step 3: Activate the Extension

Click the Image Downloader icon in the Safari toolbar (Mac) or access it through the extensions menu (iPhone). The extension will scan the current page and display all detected images.

### Step 4: Preview and Select Images

The extension shows thumbnails of every image found on the page, including:

- Visible images in the page content
- Background images loaded via CSS
- Icons and logos
- Thumbnails and full-size versions

You can select all images for bulk download or check individual images you want to keep. Most image downloaders also show the image dimensions and file size, helping you identify which versions are the highest quality.

### Step 5: Download

Click the download button to save your selected images. On Mac, they will typically save to your Downloads folder or a location you specify. On iPhone, images are saved to your Photos library or Files app.

## Tips for Better Image Downloads

**Filter by size.** Most pages contain many small icons, spacer images, and tracking pixels alongside the content images you actually want. Use size filters to exclude images below a certain dimension, like 200x200 pixels.

**Check image quality.** Some websites serve lower-resolution thumbnails on the page and load the full-size version only when you click. If you need the highest resolution, try clicking through to individual image pages before downloading.

**Organize after downloading.** When downloading images from multiple pages, create a folder structure before you start so images from different sources stay organized.

**Respect copyright.** Always respect copyright and terms of use. Many images online are protected by copyright, so only download images you have the right to use, or that are explicitly available under permissive licenses.
