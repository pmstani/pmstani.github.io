## Why Dark Mode Matters for Web Browsing

Bright white web pages can cause significant eye strain, especially during evening browsing sessions. While macOS and iOS both offer system-wide dark mode, most websites still serve blinding white backgrounds regardless of your system preference. This is where a dedicated dark mode Safari extension becomes essential.

Dark mode reduces the amount of blue light hitting your eyes, can help reduce eye fatigue during long reading sessions, and many people simply prefer the look of dark interfaces. If you have been waiting for every website to add native dark mode support, you will be waiting a long time. The better approach is to take matters into your own hands.

## Introducing Make It Dark Mode

[Make It Dark Mode](/apps/make-it-dark-mode/) is a Safari extension that applies a dark theme to any website you visit. Unlike simple color inversion filters that make images look strange and break page layouts, Make It Dark Mode uses an intelligent theme engine that understands page structure and applies dark styling that looks natural and intentional.

## How to Set Up Dark Mode for Every Website in Safari

### Step 1: Install the Extension

1. Open Safari on your Mac and navigate to **Safari > Settings > Extensions**.
2. Click **More Extensions** to open the App Store.
3. Search for "Make It Dark Mode" and install it.
4. Return to Safari Settings and enable the extension by checking its checkbox.

On iPhone, go to **Settings > Apps > Safari > Extensions**, then install and enable it from there.

### Step 2: Enable Dark Mode Globally

Once installed, click the Make It Dark Mode icon in the Safari toolbar. You will see a toggle to enable dark mode globally. Turn this on, and every website you visit will automatically get a dark theme applied.

### Step 3: Customize Your Dark Theme

Not all dark modes are created equal. Make It Dark Mode lets you adjust several settings:

- **Darkness level** — Choose how dark the background should be. Some people prefer true black (great for OLED displays), while others prefer a softer dark gray.
- **Accent colors** — Customize link and highlight colors to match your personal preference.
- **Contrast** — Adjust text contrast against the dark background for optimal readability.

### Step 4: Set Per-Site Preferences

Some websites already have excellent dark modes built in. For those sites, you might want to disable the extension so it does not conflict. Make It Dark Mode allows you to create a whitelist of sites where the extension stays inactive.

To add a site exception:

1. Visit the website you want to exclude.
2. Click the Make It Dark Mode toolbar icon.
3. Toggle off the extension for that specific site.
4. The extension will remember your preference for future visits.

### Step 5: Schedule Automatic Dark Mode

One of the most useful features is automatic scheduling. You can set Make It Dark Mode to activate at sunset and deactivate at sunrise, matching your natural viewing patterns:

1. Open the extension settings.
2. Find the scheduling section.
3. Set your preferred activation and deactivation times, or choose the sunset/sunrise option.
4. The extension will automatically toggle based on your schedule.

## Tips for the Best Dark Mode Experience

**Adjust per-site when needed.** Some websites with complex layouts or unusual color schemes might need individual tweaking. Take a moment to adjust settings on sites you visit frequently.

**Combine with system dark mode.** For the most consistent experience, enable macOS or iOS dark mode alongside Make It Dark Mode. This ensures both the browser chrome and web content use dark themes.

**Consider your display.** If you have an OLED display (like the iPhone or certain MacBook Pro models), using true black backgrounds can save battery life in addition to reducing eye strain.
