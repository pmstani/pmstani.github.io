You have an idea. You need to write it down. You open Google Drive, wait for it to load, click New, click Google Docs, wait for the blank document to load, and by the time you start typing, the thought has already started to fade. This friction is small but real, and it adds up across dozens of documents per week.

There are faster ways to create a new Google Doc. This guide covers every method — from browser shortcuts to dedicated apps — so you can go from idea to blank document in seconds.

## Method 1: Use the .new Domain Shortcuts

Google offers a set of `.new` domain shortcuts that instantly create new documents in your browser:

- **doc.new** — Creates a new Google Doc
- **sheet.new** — Creates a new Google Sheet
- **slide.new** — Creates a new Google Slides presentation
- **form.new** — Creates a new Google Form
- **site.new** — Creates a new Google Site

Simply type one of these into your browser's address bar and press Enter. A new, untitled document opens immediately. No Drive navigation, no menus, no waiting.

### Limitations of .new Shortcuts

- They only work in a web browser.
- You need to be signed into your Google account.
- You cannot choose which Google account the doc is created in if you have multiple accounts signed in — it defaults to the active one.
- There is no way to set a default folder, title, or template.

## Method 2: Google Drive Keyboard Shortcuts

If you already have Google Drive open, these keyboard shortcuts speed things up:

1. Navigate to Google Drive in your browser.
2. Press **Shift + T** to create a new Google Doc.
3. Press **Shift + S** to create a new Google Sheet.
4. Press **Shift + P** to create a new Google Slides presentation.

These shortcuts only work when you are on the Google Drive page with no file selected.

## Method 3: Use Browser Bookmarks

Create bookmarks for the `.new` URLs and pin them to your browser's bookmarks bar:

1. Open your browser and navigate to `doc.new`.
2. Bookmark the page.
3. Name it something short like "New Doc."
4. Repeat for `sheet.new` and `slide.new`.

One click on the bookmark, and your new document opens.

## Method 4: Create a Desktop or Home Screen Shortcut

### On Mac

1. Open `doc.new` in Safari or Chrome.
2. Drag the URL from the address bar to your desktop.
3. Double-click the shortcut anytime to create a new doc.

### On iPhone

1. Open `doc.new` in Safari.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Name it "New Doc" and tap Add.

Now you have a one-tap shortcut on your iPhone's home screen.

## Tips for a Faster Google Docs Workflow

### Use Templates for Repetitive Documents

If you frequently create the same type of document — meeting notes, project briefs, weekly reports — set up Google Docs templates:

1. Create a document with your standard formatting, headers, and placeholders.
2. Open the template gallery in Google Docs.
3. Submit your document as a template.

Next time, create a new doc from the template instead of starting blank.

### Name Documents Immediately

Untitled documents are a productivity trap. The moment a new doc opens, type a descriptive title. "Q1 Marketing Plan" is findable. "Untitled document (14)" is not.

### Organize with Starred Folders

Star your most-used Google Drive folders so you can quickly move new documents into the right location. This prevents the common problem of dozens of loose documents floating in your Drive root.

### Use Google Docs Offline

Enable offline access in Google Docs settings so you can create and edit documents even without an internet connection. Changes sync automatically when you reconnect.
